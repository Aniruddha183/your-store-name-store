# Your Store Name

This is your custom e-commerce store generated with the Store Builder.

## Getting Started

### Prerequisites
- Node.js (version 14 or higher)
- npm or yarn

### Installation

1. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   ```

### Development

Run the development server:
```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see your store.

### Production

Build the store for production:
```bash
npm run build
# or
yarn build
```

Start the production server:
```bash
npm start
# or
yarn start
```

## Deployment Options

### Vercel (Recommended - Free)
1. Sign up at [vercel.com](https://vercel.com)
2. Install the Vercel CLI: `npm i -g vercel`
3. Run `vercel` in this directory
4. Follow the prompts to deploy

### Netlify (Free)
1. Sign up at [netlify.com](https://netlify.com)
2. Install the Netlify CLI: `npm install -g netlify-cli`
3. Run `netlify deploy` in this directory
4. Follow the prompts to deploy

### GitHub Pages (Free)
1. Create a GitHub repository
2. Push this code to the repository
3. In repository settings, enable GitHub Pages from the `/docs` folder

## Customization
